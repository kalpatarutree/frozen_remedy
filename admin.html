GitHub Setup

To save changes to the live site, paste a GitHub Personal Access Token below. Create one at github.com/settings/tokens/new — select Fine-grained token, choose the frozen_remedy repo, and grant Contents → Read and write. Your token is stored only in this browser.

Email Setup (EmailJS)

To send confirmation emails to customers, create a free account at emailjs.com, then:
1. Add an Email Service (Gmail works great) and copy the Service ID
2. Create an Email Template — use variables {{to_email}}, {{to_name}}, {{flavor}}, {{size}}, {{quantity}}, {{price}}
3. Copy the Template ID and your account's Public Key
